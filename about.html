About Parkinson's Disease Detection

Overview

This application uses advanced machine learning algorithms to detect Parkinson's disease through voice analysis. The system analyzes voice recordings to identify patterns associated with Parkinson's disease.

How It Works

Users can upload voice recordings, and our ML models extract various acoustic features such as fundamental frequency, jitter, shimmer, and other voice characteristics. These features are then analyzed by multiple machine learning algorithms to provide accurate predictions.

Machine Learning Models

Support Vector Machine (SVM)

Uses kernel functions to find optimal decision boundaries for classification.

K-Nearest Neighbors (KNN)

Classifies based on similarity to k nearest training examples.

Random Forest

Ensemble method using multiple decision trees for robust predictions.

XGBoost

Gradient boosting algorithm optimized for performance and accuracy.

Model Performance

Our models are trained on a dataset of 195 samples with 31 features, including 23 patients with Parkinson's disease. The models achieve classification accuracy exceeding 90%, making them reliable diagnostic tools. Performance metrics include:

  • Accuracy: Overall correctness of predictions
  • Sensitivity: Ability to correctly identify Parkinson's cases
  • Specificity: Ability to correctly identify healthy cases

Dataset

The system is trained on the UCI Parkinson's Disease Dataset, which contains:

  • Total Samples: 195
  • Features: 31 voice-related features
  • Parkinson's Cases: 23
  • Healthy Cases: 172
